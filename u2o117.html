Describe which are the appropriate filetypes for websites

Appropriate filetypes for websites does depend on how you are building your website. The commonly used files include:

HTML -Hypertext Mark-up Language which is the standard markup language for creating webpages

CSS – Cascading Style Sheets. CSS describes how HTML elements are to be displayed on the website. It defines styles for you webpages including the design, layout and variations id display for different devices and screen sizes.

JPG –“Joint Photographic Experts Group”. This is best for images and photos and is pixel-based. A drawback is that it can appear fuzzy if scaled too much from its original size.

PDF – “Portable Document Format” is the the most common print format you will find and is best for documents to be read online or downloaded and opened in Acrobat. Useful for brochures, business cards, books etc.

There are many more filetypes including PNG, SVG, EPS, GIF, TIFF etc. When Building your website it is always worth researching which would be the best filetype to use, especially if your providing a lot of documents or displaying a portfolio of artwork or even sharing videos.
